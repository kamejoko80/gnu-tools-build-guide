MSYS2 has 3 shells (and toolchains):

   * MSYS2: posix emulation with msys-2.0.dll (similar to cygwin1.dll).
   * MINGW32: 32-bit Windows native
   * MINGW64: 64-bit Windows native

=> Must use MSYS2 shell to build the source code
